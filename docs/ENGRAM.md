# Engram — AI persistent memory

This repository is prepared to use **Engram** with Cursor. The configuration is in `.cursor/mcp.json`.

- **What is Engram:** an [MCP server](https://modelcontextprotocol.io/) that gives AI agents persistent memory (remember, recall, briefing).
- **Install:** `npm install -g engram` or see [Engram docs](https://www.engram.fyi/docs).
- **Use:** once the Engram CLI is installed, Cursor can use the vault for context across sessions.
